# REPlayer v0.1.0 RC1

This is the first public release candidate for REPlayer, a Windows-native Android 14 emulator and automation platform.

## Highlights

- Native WPF workbench with an embedded official Android Emulator/gfxstream display.
- APK installation, ADB and Frida workflows, network capture, and case-scoped evidence.
- Concurrent provider-backed agents with parallel observation and serialized device control.
- Fingerprint-hardened Android persona that reduces common stock-emulator defaults.
- MIT-licensed source, automated Windows builds, SHA-256 verification, and packaged third-party notices.

## Download

`REPlayer-v0.1.0-rc.1-win-x64-app.zip` is the self-contained Windows application build. Verify it with the adjacent `.sha256` sidecar before extraction.

The GitHub application archive records `runtimeIncluded=false`: the separately published, hash-marked Android runtime payload is required for a complete end-user distribution and is not embedded in this asset.

## Requirements

- 64-bit Windows 11
- Hardware virtualization enabled
- Windows Hypervisor Platform
