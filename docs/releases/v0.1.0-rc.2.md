# REPlayer v0.1.0 RC2

This release candidate supersedes RC1 before its application archive received any downloads. RC2 preserves exact .NET redistribution notices and uses curated, verified GitHub release notes.

## Highlights

- Native WPF workbench with an embedded official Android Emulator/gfxstream display.
- APK installation, ADB and Frida workflows, network capture, and case-scoped evidence.
- Concurrent provider-backed agents with parallel observation and serialized device control.
- Fingerprint-hardened Android persona that reduces common stock-emulator defaults.
- MIT-licensed source, automated Windows builds, SHA-256 verification, and packaged third-party notices.

## Compliance and provenance

- Self-contained runtime packs are pinned to .NET 9.0.17.
- Exact .NET Core Runtime and Windows Desktop Runtime licenses are included.
- The exact .NET Core Runtime third-party notice and recorded source hashes are included.
- The release body is sourced from this reviewed document rather than generated notes.

## Download

`REPlayer-v0.1.0-rc.2-win-x64-app.zip` is the self-contained Windows application build. Verify it with the adjacent `.sha256` sidecar before extraction.

The GitHub application archive records `runtimeIncluded=false`: the separately published, hash-marked Android runtime payload is required for a complete end-user distribution and is not embedded in this asset.

## Requirements

- 64-bit Windows 11
- Hardware virtualization enabled
- Windows Hypervisor Platform
