# Android customizer asset notices

## Material Design icons

The browser, files, settings, and debugging glyphs are derived from Google's Material Design Icons repository:

- https://github.com/google/material-design-icons
- License: Apache License 2.0

The glyphs are packaged as Android vector drawables and recolored monochrome for the launcher icon theme.

## Wallpaper

`premium_wallpaper.png` is derived from the user-supplied `premium.png` asset. The source file contained AVIF data despite its `.png` filename; the build asset is a standards-compliant 540×960 PNG.
